# GGJ26-Chutney
Global game jam submission 2026

### RACOON BURGLERS
Play as a Racoon with an insatiable lust for stuff, blocked only by those pesky humans that hoard it all! Run, hide and grab your way through their house as you ransack it! 

### GAME LOOP
RUN -> THINK -> STEAL/HIDE -> LEAVE

### VERBS
- Run
- Crouch
- Grab
- Jump
